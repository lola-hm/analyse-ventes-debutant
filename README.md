# analyse-ventes-debutant
Premier mini-projet d'analyse de ventes en Python (pandas, visualisation).

## Objectif du projet
Analyser un jeu de données de ventes afin de :
- calculer le chiffre d'affaires
- identifier les produits les plus vendus
- analyser l'évolution des ventes dans le temps

## Données
Le jeu de données contient les colonnes suivantes :
- date : date de la vente
- produit : nom du produit
- prix : prix unitaire
- quantite : nombre d'unités vendues

## Outils utilisés
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Analyse réalisées
- nettoyage et préparation des données
- création d'une colonne chiffre d'affaires
- calcul de KPI
- visualisations simples

## Résultats & insights
- Le T-shirt est le produit le plus vendu en volume
- Les ventes varient selon les jours
